# Gomoku Visualizer

A modern, professional visualizer for Gomoku games using **PySide6** (Qt for Python).

### Automated Script (Recommended)

```bash
# From project root
./scripts/launch_visualizer.sh

# Or specify a custom replay file
./scripts/launch_visualizer.sh path/to/your/replay.log
```

### Manual Launch

```bash
# Install dependencies
pip3 install PySide6

# Launch with replay file
python3 visualizer/run_visualizer_pyside.py replay.log

# Launch without file
python3 visualizer/run_visualizer_pyside.py
```

### Load a Replay
```bash
./scripts/launch_visualizer.sh
# or
python3 visualizer/run_visualizer_pyside.py replay.log
```
